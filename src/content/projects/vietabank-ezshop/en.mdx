## Context

VietABank ezSHOP is a mobile banking application developed by a 10-person team at HILO Group. My work focuses on turning business requirements into technical solutions while keeping the user experience aligned with the product goals.

## Contributions

- Built a reusable NFC SDK module that exposes secure native NFC capabilities to mobile banking applications.
- Integrated Flutter modules into a React Native environment so both frameworks could be used within the same product.
- Developed technical solutions from business requirements and refined the experience around project needs.
- Researched and introduced technologies aimed at improving system performance and scalability.

## Engineering focus

The project brings together cross-framework mobile integration and native device capabilities. The NFC work was designed as a reusable module, while the Flutter integration required careful coordination with the existing React Native environment.

## Scope

Repository access, production links, and product demo footage are not public because this is a commercial banking project.
